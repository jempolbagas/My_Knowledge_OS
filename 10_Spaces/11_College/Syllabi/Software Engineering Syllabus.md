## 3. Capaian Pembelajaran (CP)

### Capaian Pembelajaran Lulusan (CPL/LO)

- **CPLO3:** Mampu menguasai dasar-dasar pengetahuan dan konsep-konsep teoritis bidang informatika, yang meliputi matematika, logika, algoritma, komputasi, struktur data, pemrograman, sistem komputer dan jaringan, pengolahan data, perangkat lunak, kecerdasan buatan, teori bahasa dan Automata, Statistika, dan analisis numerik.

### Capaian Pembelajaran Mata Kuliah (CPMK)

- **CPMK351:** Mahasiswa mampu memahami konsep Process Model dalam Software Engineering.
- **CPMK352:** Mahasiswa mampu membuat rancangan aplikasi sistem cerdas menggunakan model dalam rekayasa perangkat lunak.

## 4. Deskripsi Mata Kuliah & Bahan Kajian

**Deskripsi:** Mata kuliah ini memberikan pemahaman mengenai berbagai macam Process Model (Waterfall, Prototyping, RAD, Evolutionary), Analysis Modeling, Design Model, Object Oriented Analysis and Design (OOAD), Testing Strategies, dan Software Testing Method.

**Bahan Kajian (Subject Matters):**
1. Konsep dasar rekayasa perangkat lunak.
2. Metode pengembangan perangkat lunak.
3. Pengembangan perangkat lunak berbasis obyek.

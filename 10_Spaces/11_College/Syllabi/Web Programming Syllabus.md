## 3. Capaian Pembelajaran (CP)

### Capaian Pembelajaran Lulusan (CPL/LO)

- **CPLO4:** Mampu menguasai konsep-konsep terapan bidang Informatika, yang meliputi metode-metode penyediaan dan manajemen infrastruktur cloud yang efektif dan efisien, ekstraksi dan analisis data, komputasi kecerdasan buatan, deployment perangkat lunak, dan optimasi.

### Capaian Pembelajaran Mata Kuliah (CPMK)

- **CPMK221:** Mahasiswa mampu menjelaskan arsitektur, konsep dasar, IDE atau tools untuk membangun aplikasi.
- **CPMK222:** Mahasiswa mampu mengembangkan aplikasi berbasis web.

## 4. Deskripsi Mata Kuliah & Bahan Kajian

**Deskripsi:** Mata kuliah ini mendefinisikan proses desain yang user-centered, membuat desain antarmuka, dan melakukan usability test untuk aplikasi software. Fokus utama adalah pada pengembangan teknis aplikasi web menggunakan berbagai teknologi modern.

**Bahan Kajian (Subject Matters):**
1. Konsep dasar pemrograman Web.
2. Bahasa pemrograman HTML dan CSS.
3. Bahasa pemrograman Javascript.
4. Library JQuery & Konsep AJAX.
5. Pemrograman PHP (Basic & OOP).
6. Validasi form, Cookies, dan Session.
7. Konsep MVC pada web programming framework.

# Tugas 1

## Ketentuan Tugas

Pada tugas ini, Anda diminta untuk membuat sebuah website sederhana dengan ketentuan sebagai berikut:
1. Menggunakan HTML dan CSS saja.
2. Mengimplementasikan Box Model, Flexbox, dan Grid Layout.
3. Tema Website Bebas.

Anda bebas menentukan tema website, seperti profil pribadi, toko online sederhana, landing page, blog, dan lain sebagainya.

## Rubrik Penilaian

Penilaian tugas akan didasarkan pada beberapa aspek berikut:
1. Kerapihan Code (kode tersusun dengan rapi, mudah dibaca, dan memiliki struktur yang jelas).
2. Kelengkapan Ketentuan (website telah memenuhi seluruh ketentuan yang diminta).

## Pengumpulan

Tugas dikumpulkan dalam 2 format:
1. Laporan dalam bentuk Portable Document File (\*.pdf) dengan ketentuan nama file **PPW_Kelas_01_NIM_NamaLengkap.pdf**

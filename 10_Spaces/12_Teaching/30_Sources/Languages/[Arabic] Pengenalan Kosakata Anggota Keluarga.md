# Pengenalan Kosakata Anggota Keluarga (أُسْرَتِي)

Materi Bahasa Arab Kelas 7 SMP/MTs.

## 1. Mufradat (Kosakata) - Anggota Keluarga
Berikut adalah kosakata dasar mengenai anggota keluarga (أُسْرَةٌ) dalam bahasa Arab:

| Bahasa Arab | Cara Membaca (Latin) | Arti (Indonesia) |
| :---: | :---: | :---: |
| أَبٌ | Abun | Ayah |
| أُمٌّ | Ummun | Ibu |
| أَخٌ | Akhun | Saudara Laki-laki |
| أُخْتٌ | Ukhtun | Saudara Perempuan |
| جَدٌّ | Jaddun | Kakek |
| جَدَّةٌ | Jaddatun | Nenek |
| عَمٌّ | 'Ammun | Paman (dari pihak ayah) |
| عَمَّةٌ | 'Ammatun | Bibi (dari pihak ayah) |
| خَالٌ | Khaalun | Paman (dari pihak ibu) |
| خَالَةٌ | Khaalatun | Bibi (dari pihak ibu) |
| اِبْنٌ | Ibnun | Anak Laki-laki |
| اِبْنَةٌ / بِنْتٌ | Ibnatun / Bintun | Anak Perempuan |
| أُسْرَةٌ / عَائِلَةٌ | Usratun / 'A-ilatun | Keluarga |

### Penambahan Kata Ganti Kepemilikan "Ku" (Ya' Mutakallim)
Untuk mengatakan "Ayahku", "Ibuku", kita menambahkan huruf Ya' bersukun (يْ) di akhir kata dan huruf sebelumnya dikasrahkan:
- Ayah (أَبٌ) + ku -> Ayahku (**أَبِي** - Abii)
- Ibu (أُمٌّ) + ku -> Ibuku (**أُمِّي** - Ummii)
- Kakek (جَدٌّ) + ku -> Kakekku (**جَدِّي** - Jaddii)
- Saudara laki-lakiku (**أَخِي** - Akhii)
- Saudara perempuanku (**أُخْتِي** - Ukhtii)

## 2. Qira'ah (Teks Bacaan Singkat)
Bacalah teks berikut dan perhatikan penggunaan kosakatanya!

**أُسْرَتِي (Keluargaku)**

هٰذِهِ صُوْرَةُ أُسْرَتِي. أَنَا عَلِيٌّ.
*(Haadzihi shuuratu usratii. Anaa 'Aliyyun.)*
Ini adalah foto keluargaku. Saya Ali.

هٰذَا أَبِي، اِسْمُهُ حَسَنٌ، هُوَ مُدَرِّسٌ.
*(Haadza abii, ismuhu Hasanun, huwa mudarrisun.)*
Ini ayahku, namanya Hasan, dia seorang guru.

وَهٰذِهِ أُمِّي، اِسْمُهَا فَاطِمَةُ، هِيَ طَبِيْبَةٌ.
*(Wa haadzihi ummii, ismuhaa Faathimatun, hiya thabiibatun.)*
Dan ini ibuku, namanya Fatimah, dia seorang dokter.

هٰذَا أَخِي الكَبِيْرُ، اِسْمُهُ عُمَرُ، وَهٰذِهِ أُخْتِي الصَّغِيْرَةُ، اِسْمُهَا عَائِشَةُ.
*(Haadza akhiil kabiiru, ismuhu 'Umaru, wa haadzihi ukhtiish shaghiiratu, ismuhaa 'Aaisyatun.)*
Ini kakak laki-lakiku, namanya Umar, dan ini adik perempuanku, namanya Aisyah.

أَنَا أُحِبُّ أُسْرَتِي.
*(Anaa uhibbu usratii.)*
Saya mencintai keluargaku.

## 3. Catatan Tata Bahasa Ringkas (Qawaid)
- **هٰذَا (Haadza)** : Ini (untuk laki-laki/Muzakkar) -> Contoh: هٰذَا أَبِي (Ini ayahku).
- **هٰذِهِ (Haadzihi)** : Ini (untuk perempuan/Muannats) -> Contoh: هٰذِهِ أُمِّي (Ini ibuku).
- **هُوَ (Huwa)** : Dia (laki-laki).
- **هِيَ (Hiya)** : Dia (perempuan).
